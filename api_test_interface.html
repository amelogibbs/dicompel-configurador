📖 Como Usar

  1. Teste Health: Clique em "Testar Health" para verificar se a API está respondendo
  2. Faça Login: Use email/senha para obter um token JWT válido
  3. Copie o Token: Cole o token recebido nos campos de produtos e pedidos
  4. Requisições Protegidas: Teste os endpoints protegidos com o token
  5. Veja as Respostas: As respostas JSON aparecem nos campos de resposta
